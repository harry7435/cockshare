# Project Name

## Cock Share v.0

[Demo app](https://cockshare.vercel.app/)

<br>

## Installation

설치<br>
`npm create vite@latest cockshare -- --template react`  
<br>
local 실행 방법  
`npm run dev`  
<br>
참고사항  
`.env` 파일을 루트 폴더(/cockshare)에 따로 만들어야함  
=> firebase api key 보안 유지

<br>
